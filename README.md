## これは何？
ポケモンCHAMPIONS向けの
- **タイプ相性補完ビルダー**（`index.html`）
- **選出サポート**（`battle.html`）

を、**PC/スマホでローカル利用**できるようにしたリポジトリです。

---

## スマホでもローカルで使う（同一Wi‑Fi）
PC側でサーバーを起動して、スマホからPCのIPにアクセスします。

### 1) PCで起動

```bash
cd server
npm install
npm run start
```

起動ログに `Running on http://0.0.0.0:3000` が出ます。

### 2) スマホから開く
同じWi‑Fiに繋いだスマホで、ブラウザから
- `http://<PCのIPアドレス>:3000/index.html`
- `http://<PCのIPアドレス>:3000/battle.html`

を開いてください。

---

## スマホで“アプリ化”してオフラインでも開く（推奨）
このプロジェクトは **PWA** 対応です（`manifest.webmanifest` + `sw.js`）。

- **GitHub Pages** など **HTTPS** で公開して開く
- ブラウザの「ホーム画面に追加」

すると、**初回ロード後はオフラインでも起動**できます。

※注意: オフライン時に未取得のポケモンデータが必要になるケース（初回ロード直後など）では動作しません。いったんオンラインで読み込み完了させてください（データは `localStorage` にキャッシュされます）。

---

## GitHubに上げるときの注意
- `node_modules` は巨大なので **Git管理しません**（`.gitignore` 済み）
- `server/settings.json` は実行時に作られるので **コミットしません**
